Actual Plays

The following are my most notable APs in chronological order. Some are a bit rough around the edges and none of them have been completed, but I still think they're entertaining and may be educational.

  • Ghost - My very first solo game, using Cortex Plus.
  • Call of Cthulhu - Playing through the Corbitt House pre-made adventure in Call of Cthulhu 7e.
  • Calypso - A test run of the Calypso solo game engine.
  • Vampire - A game about vampires using Motif and Cortex Prime.
  • X-Men - A attempt at creating the X-Men using Fate Condensed. I also have a Cortex Prime version for comparison using pre-made sheets taken from various sources.
  • Worm - A game based in the Wormverse using Starforged.
  • City of Mist - A short test run of the City of Mist system using the Gambling with Death pre-made adventure.

Tools & Guides

Various tools and information related to solo roleplaying.

  • Style Guide - This is how I currently format my actual plays.
  • Markdown Converter - This is my tool for converting my actual plays written in markdown to HTML files.
  • Finola RPG - This is my app that for playing solo. It's also available in android and windows versions.
  • Solo Roleplaying Toolkit - This is my old abandoned app for playing solo. It's also available in android and windows versions.
  • Roll Characters - A simple tool to roll a bunch of gamebook characters at once. Download the file and view the source for instructions on how to add more systems.
  • City of Mist Notes - These are notes that I took while learning City of Mist, which is a great game but has a lot of moving parts and can be confusing.
  • Story Guide - Some notes on how to craft a story narrative.
